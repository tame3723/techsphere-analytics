# Analytics Engine - Step-by-Step Walkthrough

## What Does This Engine Do?

This engine analyzes 38,290 job postings across 7 tech domains to answer one question: **"Which tech career has the best opportunity right now?"**

It weighs original/real data more heavily than AI-generated data, extracts skills using NLP, and produces a final opportunity score (0-1) for each domain.

---

## Step 1: Load Data & Apply Quality Weights

**What happens:** The engine reads `final_refined_jobs.csv` and adds trust scores to each record.

**Why it matters:** Not all data is equal. Real job descriptions (weight=1.0) are more reliable than AI-generated ones (weight=0.5-0.7).

```
Original job description → weight = 1.0 (100% trust)
AI-generated description → weight = 0.6 (60% trust)
AI-generated salary → weight = 0.5 (50% trust)
```

---

## Step 5: Extract Keywords from Descriptions (Parallel NLP)

**What happens:** Uses spaCy to extract lemmatized keywords from job descriptions. Runs across 7 domains simultaneously.

**Performance:** First run ~12 seconds, cached runs ~0.5 seconds.

**Output:** `top_keywords_spacy.csv`

## Step 6: Calculate Opportunity Score & Relative Scores (0-10)

**What happens:** Combines demand, salary, diversity, safety, AI resilience, and layoff exposure into a composite score.

**The Formula:**

```
STEP 1: Normalize each metric to 0-10 scale
STEP 2: Calculate composite score (average of all metrics)
STEP 3: Rank domains by composite score
```

## Complete Pipeline Summary

```
Step 1: Load Data (38,290 records, 7 domains)
    ↓
Step 2: Apply Quality Weights (original=1.0, generated=0.5-0.7)
    ↓
Step 3: Calculate Domain Demand (job counts + market share)
    ↓
Step 4: Calculate Salary Stats (weighted averages)
    ↓
Step 5: Extract Skills & Diversity (unique skills per domain)
    ↓
Step 6: Extract Keywords (parallel NLP, 892 keywords)
    ↓
Step 7: Calculate Opportunity Score & Relative Scores (0-10 scale)
    ↓
Step 8: Generate Layoff Heatmap (4-quadrant classification)
    ↓
OUTPUT: 10 CSV files + JSON + Excel in data/analytics/
```

|-------|---------|
